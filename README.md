# Создание сайта на CMS Joomla с базой данных PostgreSQL

С помощью этого руководства вы развернете сайт на базе [CMS Joomla](https://ru.wikipedia.org/wiki/Joomla!) с базой данных под управлением [СУБД PostgreSQL](https://yandex.cloud/ru/docs/managed-postgresql/) в инфраструктуре Yandex Cloud. Сайт будет размещен на веб-сервере [Apache HTTP-сервер](https://ru.wikipedia.org/wiki/Apache_HTTP_Server), установленном на [виртуальную машину](https://yandex.cloud/ru/docs/compute/concepts/vm) под управлением ОС [Ubuntu 24.04 LTS](https://yandex.cloud/ru/marketplace/products/yc/ubuntu-24-04-lts). Для управления доменом будет использоваться сервис [Yandex Cloud DNS](https://yandex.cloud/ru/docs/dns/), а TLS-шифрование трафика будет обеспечено с помощью [TLS-сертификата](https://yandex.cloud/ru/docs/certificate-manager/concepts/managed-certificate.html), выпущенного в сервисе [Yandex Certificate Manager](https://yandex.cloud/ru/docs/certificate-manager/).

Подготовка инфраструктуры для сайта на CMS Joomla с базой данных PostgreSQL с помощью Terraform описана в [практическом руководстве](https://yandex.cloud/ru/docs/tutorials/security/alb-with-ddos-protection), необходимые для настройки конфигурационные файлы `joomla-postgresql-terraform.tf` и `joomla-postgresql-terraform.auto.tfvars` расположены в этом репозитории.
